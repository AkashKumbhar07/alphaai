# AuraMind AI

Intelligent Crypto Market Analysis & Decision Engine.

**Stack:** Svelte + Go + Wails (Desktop-First) | gRPC | Python AI Services

## Quick Start

```bash
git clone https://github.com/AkashKumbhar07/auramind.git
cd auramind
bash tools/auractl.sh
```

Select option **16** (Setup Dev Environment), then **1** (Run Desktop App).
